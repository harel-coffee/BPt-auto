BPt.BPtEvaluator.train_subjects¶

property BPtEvaluator.train_subjects¶
This parameter stores the training subjects / index used in every fold of the cross-validation. It can be useful in some cases to check to see exactly what cross-validation was applied.
This parameter differs from all_train_subjects in that even subjects with missing target values are not included.
