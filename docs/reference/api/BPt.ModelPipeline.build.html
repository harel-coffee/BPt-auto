BPt.ModelPipeline.build¶

ModelPipeline.build(dataset, problem_spec='default', **extra_params)[source]¶

This method generates a sklearn compliant estimator version of the current Pipeline with respect to a passed dataset and Dataset and ProblemSpec.

This method calls get_estimator() with pipeline set as itself, see get_estimator() for parameters.
