Data Files#

Data files allow BPt to work with any type of arbitrary data beyond simply tabular data. For example, Data Fileâ€™s can be used to load and work with volumetric or surface based neuroimaging data or even connectome data.

See add_data_files for how to load data files to a Dataset.

During construction of a model pipeline any loaded data files must have a corresponding Loader where those files will be converted into valid down stream predictive features.
